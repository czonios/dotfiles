### Setup
  * open a terminal in your home folder (~)
  * clone this repo
  * install alacritty
  * install nvim
  * install tmux
  * install starship
  * install FiraCode Nerd Font
  * move all files from ~/dotfiles into ~ `find . -mindepth 1 -maxdepth 1 -exec mv -t .. -- {} +`
  * `git submodule init`
  * `git submodule update`

vivado launchers go into /usr/share/applications/
